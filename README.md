# AI Children Bedtime Story Generator

An AI-powered bedtime story web application built using React, Vite, Node.js, and Express.

The application generates:
- AI bedtime stories
- Dynamic story illustrations
- Story continuation
- Calm child-friendly storytelling experience

---

# Features

- AI-generated bedtime stories
- Dynamic illustration system
- Story continuation
- Child-safe themes and characters
- Calm narration support
- Responsive UI
- Local illustration fallback system
- React + Express full-stack architecture

---

# Tech Stack

Frontend:
- React
- Vite
- CSS

Backend:
- Node.js
- Express.js

AI APIs:
- Groq API (Story generation)

# Installation Steps

## 1. Clone or Download Project

Extract the project folder.

---

## 2. Open Terminal

Navigate to project folder:

```bash
cd AI_Children_bedtimestory


## 3. Install Dependencies
```bash
npm install


## 4.Environment Variables

Create a .env file in the root folder.

Example:
</>env
OPENAI_API_KEY=

OPENAI_MODEL=gpt-4.1-mini

OPENAI_IMAGE_MODEL=gpt-image-1

PORT=5050

AI_PROVIDER=groq

GROQ_API_KEY=your_groq_api_key

GROQ_MODEL=llama-3.1-8b-instant


##4. Running the Project

Start frontend and backend together:

```bash
npm run dev
